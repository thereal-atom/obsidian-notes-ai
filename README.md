# obsidian ai web

upload your markdown files and interact with your notes via a Large Language Model

built with:
- Next.js (create-t3-app)
- Tailwind CSS
- TypeScript
- Zod
- tRPC
- Supabase + supabase-js
- Google Generative AI (Gemini and Text Embedding)
